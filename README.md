MLBB Oracle — это веб-чат с искусственным интеллектом, который отвечает на вопросы об игре Mobile Legends: Bang Bang. Программа не требует интернета для работы ответов — вся база знаний по героям, билдам, мете, ротациям, эмблемам и контрпикам встроена прямо в код. Проект полностью работает в браузере, сохраняет историю диалога и аватар пользователя.
