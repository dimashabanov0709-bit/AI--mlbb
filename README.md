MLBB Oracle — это веб-чат с искусственным интеллектом, который отвечает на вопросы об игре Mobile Legends: Bang Bang. Программа не требует интернета для работы ответов — вся база знаний по героям, билдам, мете, ротациям, эмблемам и контрпикам встроена прямо в код. Проект полностью работает в браузере, сохраняет историю диалога и аватар пользователя.

Как пользоваться

Скачайте или склонируйте репозиторий, затем откройте файл index.html в любом браузере.

Введите свой никнейм — он сохранится автоматически.

Нажмите кнопку "Загрузить фото", чтобы поставить свою аватарку (она появится рядом с вашими сообщениями).

Задайте вопрос в поле ввода или выберите один из быстрых чипов (например, "Мета герои", "Fanny гайд", "Билд Лейлы").

Нажмите Enter или кнопку отправки — AI даст развёрнутый текстовый ответ без лишних эмодзи.

История чата сохранится даже после закрытия и перезагрузки страницы. Кнопка "Очистить" удаляет всю переписку.

Никакой установки, серверов или регистрации не требуется. Просто откройте index.html и начинайте спрашивать о Mobile Legends.
